## Requirements

- python == 3.6
- torch == 1.7.0+cu110
- networkx == 2.4
- gensim == 3.8.1

See [requirements.txt](https://github.com/YihongT/DASTNet/blob/master/requirements.txt) for more details.

## Dataset

- PeMS04
- PeMS07
- PeMS08

Please refer to [STSGCN (AAAI2020)](https://github.com/Davidham3/STSGCN).
